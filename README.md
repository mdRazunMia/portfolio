# Personal Portfolio Platform

Phase-by-phase implementation based on the portfolio specification:

- `frontend`: Next.js app for the public portfolio and admin UI
- `backend`: NestJS API for portfolio content and admin authentication

## Phase 1

Current implementation includes:

- Monorepo workspace structure
- NestJS backend foundation
- JWT-based admin auth scaffold
- `profile`, `projects`, and `experience` modules
- Frontend placeholder app structure for the next phase

## Next steps
